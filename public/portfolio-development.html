Designing & Developing my Portfolio

Iterative Design Process for my Website



Early iterations were based around a single-page website with minimal content, my aim being to create a clean and refined look. During the third iteration I decided to have individual pages for each work example, providing more space to detail my design process and include additional images.

I wanted the portfolio to reflect my own style and favoured aesthetic of minimalism. As with most of my designs, I focused on leaving a large amount of white space to draw the user's attention to the important content.

By working iteratively, I was able to test various colour schemes, typography and layouts until I settled on the current styles. I used GitHub throughout this process for version control and to gain experience using git on the OS X command line. By overriding and customising Bootstrap styles I learnt more about developing responsive websites.
